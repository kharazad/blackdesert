관리자 모드

데이터베이스(Firebase) 설정이 필요합니다

텍스트를 웹에서 직접 수정하고 저장하는 기능을 사용하려면, 데이터를 저장할 수 있는 Google Firebase 연동이 필수입니다. 현재 소스코드 내부의 설정값이 비어있어 로그인이 차단되었습니다. 아래 3단계를 따라 설정해주세요.

1단계: Firebase 프로젝트 생성하기

1. Firebase Console (클릭)에 접속하여 [프로젝트 만들기]를 진행합니다.
2. 프로젝트가 생성되면 화면 중앙의 웹() 아이콘을 클릭하여 앱을 등록하고, 화면에 나타나는 firebaseConfig 코드를 복사합니다.

2단계: 소스코드에 붙여넣기

index.html (그리고 공성전 가이드 파일) 소스코드를 메모장 등으로 엽니다. 맨 아래쪽 스크립트 영역에 있는 localFirebaseConfig 부분을 방금 복사한 본인의 구성값(apiKey, projectId 등)으로 교체하여 깃허브에 다시 업로드합니다.

3단계: 로그인/데이터베이스 사용 설정

1. Firebase 좌측 메뉴에서 Authentication을 누르고 '시작하기'를 누른 뒤, Google 로그인과 익명 로그인 제공업체를 추가(사용 설정)합니다.
2. 좌측 메뉴에서 Firestore Database를 누르고 '데이터베이스 만들기'를 선택한 뒤 [테스트 모드]로 데이터베이스를 생성합니다.
